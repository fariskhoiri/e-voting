
# Aplikasi E-Voting berbasis Web PHP Native

Berikut ini adalah sebuah project aplikasi pemilihan umum berbasis web yang dibuat menggunakan PHP Native tanpa framework sama sekali.


## Sekilas info

Ini adalah project kecil saya di semester 7 untuk memenuhi tugas akhir mata kuliah Pemrograman Web 2.
## Installation

Pastikan gunakan php versi 8.x.x keatas

- Cek versi php dengan mengetikkan di terminal:

```bash
  php -version
```
- Clone atau download repositori ini:

```bash
  git clone https://github.com/fariskhoiri/e-voting.git
```
- Letakkan folder project (jika menggunakan XAMPP) ke direktori komputer lokal:

```bash
  \xampp\htdocs
```
- Atau jika menggunakan Laragon ke direktori komputer lokal:

```bash
  \laragon\www
```

- Jalankan Web Server dan DBMS (MySQL) pada XAMPP atau Laragon.

- Login ke phpmyadmin lalu impor basis data dari folder database/pemilu_db.sql ke MySQL.

- Akses aplikasi melalui:

```bash
  http://localhost/e-voting-main/index.php
```

